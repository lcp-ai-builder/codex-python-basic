# Codex Python Basic

Starter Python project initialized for VS Code usage. Includes basic project structure, lint/test tooling via `pip`, and helpful VS Code configuration files.

## Getting Started

1. Create a virtual environment: `python -m venv .venv`
2. Activate it: `source .venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the sample app: `python -m src.app`
5. Run tests: `pytest`

Adjust project metadata inside `pyproject.toml` as needed.
